Perguntas

  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

      2. R= Para fornecer uma dica ou descrição

  2. Na tag <input> com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

      3. R= Definir o estado inicial de uma opção selecionável

  3. Na tag <input> para que serve o atributo "value"?

      4. R= Definir um valor pré-definido para um campo de entrada e enviar esse valor junto com os dados do formulário quando ele é enviado.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

      5. R= Agrupar botões de opção em um formulário

  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

      6. R= É usado para definir qual opção deve ser selecionada por padrão em um elemento <select> quando a página é carregada.

  6. Na tag <option> para que serve o atributo "selected"?

      7. R= Tag de formulário que é usado para garantir que um campo de entrada seja preenchido antes que o formulário possa ser enviado

  7. Nas tags de formulário para que serve o atributo "required"?

      8. R= Indica o URL do script ou arquivo que processa os dados enviados pelo formulário quando ele é submetido.

  8. Na tag <form> para que serve o atributo "action"?

      9. R= Ele define o URL ou caminho do arquivo no servidor que será responsável por processar os dados do formulário

  9. Na tag <form> para que serve o atributo "method"? Quais são os valores possíveis desse atributo?

      10. R= Especificar o método HTTP que será usado para enviar os dados do formulário para o servidor

  10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form> ?

      11. R= O valor do atributo "method" na tag <form> deve ser definido como "POST"

  11. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

      12. R= O valor que deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos é "POST".

  12. O que as tags <fieldset> e <legend produzem?

      13. R= São usadas em conjunto para criar grupos de campos relacionados em um formulário
